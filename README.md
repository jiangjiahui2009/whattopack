# whattopack
隐私政策

  生效日期：2026年5月10日

  "带什么"App（以下简称"本App"）高度重视用户隐私。本隐私政策说明了我
  们如何收集、使用和保护您的信息。

  一、信息收集

  本App不收集任何个人信息。

  - 不要求注册账号
  - 不收集姓名、手机号、邮箱等个人身份信息
  - 不收集位置信息
  - 不收集设备信息
  - 不接入任何第三方统计、分析或广告SDK

  二、数据存储

  您创建的所有清单数据均存储在您设备的本地文件系统中，以 Markdown
  文件形式保存。本App不会将您的数据上传至任何服务器，您拥有数据的完全
  控制权。

  删除App时，相关数据将一并从您的设备中移除。

  三、相册访问

  本App仅在您主动使用"保存分享图到相册"功能时，请求访问您的相册以写入
  图片。该权限仅用于此单一目的，不会读取相册中的其他内容。
